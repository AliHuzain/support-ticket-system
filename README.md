# Support Ticket System

A simple Python project that demonstrates basic ticket creation and status updates.

## Features
- Create a support ticket (title, description)
- Track ticket status (Open / Closed)
- Simple, clean project structure

## Project Structure
- `app/main.py` — runs a small demo
- `app/tickets.py` — Ticket class and logic

## How to Run
```bash
python app/main.py
